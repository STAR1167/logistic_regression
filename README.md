# Heart Disease Prediction using Logistic Regression (NumPy)
## Overview

This project implements a Heart Disease Prediction system using Logistic Regression built entirely from scratch with NumPy.

The objective is to predict whether a patient is likely to have heart disease based on clinical parameters.

This project focuses on understanding the mathematical foundations of logistic regression, including:

- Sigmoid function
- Maximum Likelihood Estimation (MLE)
- Binary Cross-Entropy Loss
- Gradient Descent Optimization
- Model Evaluation Metrics

No machine learning frameworks such as Scikit-learn were used for model training.
## Dataset

Dataset: UCI Heart Disease Dataset

Features:

- age
- sex
- cp (chest pain type)
- trestbps (resting blood pressure)
- chol (serum cholesterol)
- fbs (fasting blood sugar)
- restecg (resting electrocardiographic results)
- thalach (maximum heart rate achieved)
- exang (exercise-induced angina)
- oldpeak (ST depression induced by exercise)
- slope (slope of peak exercise ST segment)
- ca (number of major vessels)
- thal (thalassemia)

Target:

- 0 → No heart disease
- 1 → Heart disease present
